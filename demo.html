px-navbar

The px-navbar enables navigation through an information hierarchy and, optionally, management of screen contents.

When the user goes to a new level in a navigation hierarchy, two things should happen:

  1. The navigation bar title should change to the new level’s title, if appropriate.
  2. A back button should appear in the left end of the bar; it can be labeled with the previous level’s title if it adds value.

In general, a navigation bar should contain no more than the view’s current title, the back button, and one control that manages the view’s contents.

By default, the buttons are a minimum size of 44x44 (Per Apple Design Guidelines)
